--Очень кратко - 3 секунды:-------------------------------------------------
Идея - Проект - Задачи - Журнал - Ресурсы - Источники

--Коротко - 30 секунд:------------------------------------------------------
Идея в программе хранится как проект, проект делится на задачи, при выполении каждой задачи ведется журнал.
 
Для выполнения задачи подгружаются разные файлы или ссылки на сайты(ресурсы). 

Ресурсы = ссылки на источники(в таблице с всеми файлами/ссылками и т.п.).
 
Т.е. один и тот же ресурс может использоваться в разных задачах. 

При добавлении источника создается папка с именем проекта с вложенными папками задачами,

в которых хранятся файлы используемые в данных задачах.
